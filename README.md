# Setting Up

## Pre-requisites for Front-end development

- node v16
- npm

## Pre-requisites for Back-end development

- Python v3.11
- Poetry
- pyenv

## Installation


1. Install node requirements
```
npm install
```
2. Run 
```
npm run dev
```
